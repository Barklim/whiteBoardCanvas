## Whiteboard Canvas

Whiteboard with basic cores functions figma like. [link](https://barklim.github.io/whiteBoardCanvas/)

```
ctrl + mouse - zome
scroll wheel - mouse wheel
drag by mouse clicking
```



### Prerequisite

- Node.js - [14.17.0](https://nodejs.org/en/blog/release/v14.17.0)

### Scripts

```
npm install
npm run start 
```
